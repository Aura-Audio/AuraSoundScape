# AuraSoundScape v4.0

**A modular browser-based generative audio engine for musicians, sound designers, and experimenters.**

## Features
- Polyphony
- FM Synthesis
- Granular Synthesis
- Wavetable Synthesis
- Sample Playback
- MIDI Input
- Arpeggiator
- Sequencer
- LFOs
- Envelope Followers
- Audio Input

## Quick Start
1. Clone the repository
2. Open `index.html` in a browser
3. Or run `npm start`
